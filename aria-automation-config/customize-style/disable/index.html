    Disable features

    You can disable feature in docdock by changing some params in config.toml

    hide Next / Prev Chevrons

    [params]
disableNavChevron = true

    hide Search box in side menu

    [params]
disableSearch = true

    hide the icon in side bar

    [params]
disableHomeIcon = true
